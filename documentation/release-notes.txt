json2puml v1.1.8 - 09.04.2022
----------------------------------------------------------------------------------------------------------------------------------------
New Features
- New configuration parameter "groupDetailObjectsTogether" default : false
  This parameter allows to generate "together" objects in plantuml to group the detail objects to their parent objects. PlantUML then tries
  to draw the grouped objects nearer together and not to distribute them. Sometimes based on the grouping PlantUML generates duplicate 
  lines which made the diagrams hard to read. Based on that the parameter is introduced to make this optional.
- New configuration parameter "hideDuplicateRelations" default : true
  The parameter allows to hide relationships between objects when the parameter "from object", "to object",  "relationshipTypeProperty" and  
  "replationshipType" are identical.
  
Changed Features  
- In the definition file the option detaillist are also stored when the Operation is Replace and the list is empty. This allows to overwrite 
  a list with an empty list.
